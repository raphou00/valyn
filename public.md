# Valyn Public Website Specification

## Goal

Build the public marketing website for Valyn, a Shopify app that automates customer support for order tracking and repetitive customer questions.

The website must:

- Explain the product clearly
- Convert visitors into Shopify app installs
- Build trust
- Support SEO acquisition
- Provide legal/compliance pages
- Prepare for Shopify App Store review
- Support Google Ads and organic traffic

# FAQ Page `/faq`

## Questions

### What does Valyn do?

Valyn automates repetitive Shopify support emails, starting with order tracking requests.

### Does Valyn replace my helpdesk?

No. Valyn is focused on repetitive order support. It is not a full helpdesk.

### Does Valyn modify my Shopify store?

No. The first version should only read order-related data required to answer customer questions.

### What happens if Valyn cannot find an order?

It sends a fallback reply or marks the message for manual review.

### Can I disable automation?

Yes. Merchants can pause automation at any time.

### Does Valyn support multiple languages?

Planned/available depending on version: English, French, German.

### Is customer data secure?

Yes. Explain token security, limited access, and data minimization.

### Is Valyn useful for dropshipping stores?

Yes, especially stores receiving many repetitive tracking questions.
